# Лаборатория изучения C++ 🚀

Добро пожаловать в мой репозиторий для изучения языка программирования C++!
Здесь собраны материалы, примеры кода и упражнения по указателям и ссылкам.

## 📚 Структура репозитория

```
cpp_pointers_mastery/
│
├── 01_memory_basics/          # Основы памяти и адресов
├── 02_pointers_basic/         # Базовые указатели
├── 03_references_basic/       # Базовые ссылки
├── 04_pointers_and_arrays/    # Указатели и массивы
├── 05_function_parameters/    # Параметры функций
├── 06_dynamic_memory/         # Динамическая память
├── 07_function_pointers/      # Указатели на функции
├── 08_const_and_pointers/     # Константность и указатели
├── 09_smart_pointers/         # Умные указатели (C++11)
├── 10_advanced_topics/        # Продвинутые темы
├── common/                    # Общие утилиты
├── projects/                  # Учебные проекты
└── README.md                  # Этот файл
```

## 🔧 Требования

- Компилятор с поддержкой C++17 и выше
- CMake (опционально)
- Git (для контроля версий)

## 📝 Как пользоваться

1. Изучайте темы по порядку (от 01 до 10)
2. В каждой папке есть README с теорией
3. Запускайте примеры и экспериментируйте
4. Выполняйте упражнения для закрепления

## 🚀 Быстрый старт

```bash
git clone [url-репозитория]
cd cpp_lab
g++ 01_memory_basics/01_address_of_variable.cpp -o test
./test
```
